# 电气系统

本文档详细说明FRC 6353机器人的电气设计和接线。

## 电源系统

### 主电池
- **类型**: Lithium-ion 18650
- **电压**: 12.6V (10S配置)
- **容量**: 100Ah
- **最大放电**: 200A

### 电源分配板 (PDP)
```
┌──────────────────┐
│   12V电池        │
│   (100Ah)        │
└────────┬─────────┘
         │
    ┌────▼────┐
    │   PDP   │
    │(CTRE)   │
    └────┬────┘
    ┌────┴────────┬──────┬──────┐
    │    │        │      │      │
  ┌─▼─┐ │  ┌────┐│┌────┐│┌────┐│
  │PCM│ │  │电机│││传感││└────┘│
  └───┘ │  │    ││      │
        │  └────┘│└────┘│
        └────────┴──────┘
```

## 控制器

### RoboRIO
- **处理器**: ARM Cortex-A9, 667MHz
- **内存**: 128MB DRAM
- **输入/输出**: 10个模拟输入, 16个数字I/O
- **通信**: CAN, I2C, SPI, UART

### 电机控制器

| 电机类型 | 控制器 | 通道 | 用途 |
|---------|--------|------|------|
| Falcon 500 | Talon SRX | CAN | 底盘驱动 |
| NEO | PWM | 1,2,3,4 | 发射器 |
| 775Pro | Spark Max | PWM | 进料系统 |

## CAN总线拓扑

```
RoboRIO
  ▲
  │ (CAN)
  ▼
┌─────────────────────┐
│  Talon SRX (ID:1-6) │  ← 底盘电机
│  Talon SRX (ID:5)   │  ← 发射电机
│  PDP (ID:0)         │
└─────────────────────┘
```

## 传感器配置

### 编码器
- **类型**: Integrated Falcon Encoders
- **位置**: 每个驱动电机
- **分辨率**: 2048 ticks/rev

### 陀螺仪
- **型号**: NavX-MXP
- **通信**: SPI or I2C
- **用途**: 机器人方向控制

### 距离传感器
- **型号**: LIDAR-Lite v3
- **范围**: 0-40m
- **I2C地址**: 0x62

## 接线规范

### 重要规则
1. ✓ 红色线 = 正(+12V)
2. ✓ 黑色线 = 负(GND)
3. ✓ 绿色线 = CAN High
4. ✓ 白色线 = CAN Low
5. ✓ 所有CAN接线端接120Ω电阻

### 连接器类型
- 电源: Anderson SB50
- CAN: M12 X-Coded or 3-pin Weidmüller
- PWM: 3-pin JST
- I2C: 4-pin JST

## 故障诊断

| 症状 | 可能原因 | 解决方案 |
|------|--------|--------|
| 红光闪烁 | 电池电压低 | 充电电池 |
| 电机无响应 | CAN连接断开 | 检查CAN线 |
| 传感器读数错误 | I2C冲突 | 检查地址设置 |

## 安全检查清单

- [ ] 电池连接牢固
- [ ] 所有接触点无腐蚀
- [ ] CAN总线端接正确
- [ ] 没有裸露的接线
- [ ] 电压在12V±0.5V范围内
