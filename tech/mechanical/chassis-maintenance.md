# 底盘维修指南

底盘是机器人的基础，定期维护和检查非常重要。

## 定期检查项目

### 视觉检查 (每次使用后)
- [ ] 检查所有螺钉是否松动
- [ ] 查看底盘是否有裂纹
- [ ] 检查轮子是否磨损或损坏
- [ ] 确认传送带张力正确

### 功能测试 (每周)
- [ ] 测试所有4个车轮是否正常转动
- [ ] 检查前进、后退、转弯功能
- [ ] 验证编码器读数准确
- [ ] 测试差速驱动平衡性

### 深度检查 (每月)
- [ ] 清洁所有运动部件
- [ ] 检查轴承状态
- [ ] 验证电机连接牢固
- [ ] 检查CAN总线连接

## 常见问题和解决方案

### 问题1: 某个轮子不转动

**原因**:
- 电机断开连接
- CAN总线故障
- 电机损坏
- 齿轮卡死

**解决步骤**:
1. 检查电源连接
2. 验证CAN ID匹配
3. 测试电机是否接收信号
4. 检查齿轮或传动是否卡死
5. 必要时更换电机

### 问题2: 底盘运动方向错误

**原因**:
- 电机接线反向
- 代码中的倒序配置
- 编码器反向

**解决步骤**:
1. 检查接线顺序
2. 验证WPILib代码中的电机设置
3. 确认编码器方向配置

### 问题3: 轮子打滑

**原因**:
- 轮胎磨损
- 传送带松动
- 地面太光滑
- PID控制不当

**解决步骤**:
1. 检查轮胎纹理
2. 调整传送带张力
3. 调整PID参数
4. 必要时更换轮胎

## 维护及时表

| 任务 | 频率 | 预计时间 | 所需工具 |
|------|------|--------|--------|
| 清洁检查 | 每周 | 15分钟 | 布, 刷子 |
| 螺钉紧固 | 每月 | 30分钟 | 螺丝刀, 扳手 |
| 轴承润滑 | 每月 | 20分钟 | 润滑油 |
| 深度检查 | 每季度 | 2小时 | 完整工具箱 |

## 安全提示

⚠️ **重要**:
- 维修前关闭所有电源
- 断开电池连接
- 禁用无线指令
- 使用适当的工具和个人防护装备
- 需要时向技术负责人寻求帮助

- Falcon 500电机 (2个)

- 各种规格螺钉和螺帽
- 轴承 (若干)
