电机更换指南 ​

当电机损坏或性能下降时，需要进行更换。本指南提供详细的电机更换步骤。

准备工作 ​

所需工具和材料 ​

工具:

  • 十字螺丝刀组
  • 梅花扳手组
  • 六角扳手
  • 万用表
  • 焊接设备 (如需要)

材料:

  • 替换电机 (适当型号)
  • 新螺钉/螺帽 (如旧的已損損)
  • 电气胶带
  • 导热膏 (可选)

安全предупреждение ​

⚠️ 必须遵守:

  • 关闭所有电源
  • 断开电池连接
  • 放电RoboRIO
  • 禁用无线系统
  • 佩戴防护眼镜
  • 需要时向导师寻求帮助

电机类型识别 ​

Falcon 500 (CTRE) ​

识别特征:

  • 黑色外壳，红色配件
  • 集成编码器和制动器
  • CAN连接
  • 规格: 500W, 2048 ticks/rev

更换难度: ⭐⭐ (中等)

所需时间: 20-30分钟

NEO 550 (REV) ​

识别特征:

  • 蓝色配件
  • 小型设计
  • PWM或CAN连接可选
  • 规格: 262W

更换难度: ⭐ (简单)

所需时间: 15-20分钟

775Pro (AndyMark) ​

识别特征:

  • 传统圆形铁壳
  • PWM连接
  • 齿轮箱兼容
  • 规格: 300W

更换难度: ⭐⭐ (中等)

所需时间: 15-20分钟

快速更换流程 ​

第1步: 断开连接 ​

电气连接:

如果是PWM电机:
1. 拆下PWM电缆
2. 记录接线或拍照留档
3. 从控制器拔出

如果是CAN电机:
1. 记录CAN ID
2. 从CAN总线断开
3. 记录配置参数

机械连接:

1. 移除驱动轴上的螺钉
2. 拆下安装支架
3. 从机器人上卸下电机

第2步: 安装新电机 ​

机械安装:

  1. 对齐新电机与安装孔
  2. 使用对应螺钉固定
    • 不要过紧 (可能损坏电机)
    • 确保对齐正确
  3. 重新安装驱动轴

检查对齐:

旋转轴可以:
✓ 自由转动
✓ 无明显偏心
✓ 无卡顿感

第3步: 重新连接电气 ​

PWM连接:

  1. 将红线连接到+5V
  2. 将黑线连接到GND
  3. 将信号线连接到PWM引脚
  4. 验证接线

CAN连接:

  1. 连接CAN H和CAN L
  2. 设置正确的CAN ID
    • 使用CTRE Tuner或REV Hardware Client
    • 避免ID重复
  3. 测试通信

第4步: 验证和测试 ​

软件配置更新:

java
// 如果更改了CAN ID，更新代码
private TalonFX leftMotor = new TalonFX(NEW_MOTOR_ID);

// 或如果没有CAN，检查PWM端口
private PWMSparkMax motor = new PWMSparkMax(PWM_PORT);

功能测试:

测试清单:
✓ 电机正常启动
✓ 方向正确 (如需反向，调整代码而非接线)
✓ 速度响应正常
✓ 温度正常
✓ 编码器读数准确 (如适用)

常见问题 ​

问题1: 新电机不响应 ​

排查步骤:

  1. 检查电源连接
  2. 验证CAN ID是否正确
  3. 检查代码中的电机ID
  4. 使用万用表测试电压
  5. 检查控制器是否损坏

问题2: 电机方向反向 ​

解决方案:

java
// 软件反转 (推荐)
leftMotor.setInverted(true);

// 不要直接反转接线，除非必要

问题3: CAN总线错误 ​

检查:

  1. 确认所有CAN连接牢固
  2. 验证总线端接正确 (120Ω电阻)
  3. 使用CTRE Tuner诊断总线
  4. 检查是否有CAN ID冲突

问题4: 编码器读数不正确 ​

对于Falcon 500:

1. 在CTRE Tuner中清除编码器零位
2. 旋转轴向前，验证正值
3. 使用setSelectedSensorPosition(0)重置

对于NEO带编码器:

1. 在REV Hardware Client中检查编码器
2. 验证位置和速度读数
3. 必要时更换编码器
