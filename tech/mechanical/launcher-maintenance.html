发射器维修指南 ​

发射系统是FRC机器人中最复杂的部分之一，需要定期维护以确保最佳性能。

日常检查 ​

赛前检查 (每场比赛前) ​

  • [ ] 检查飞轮是否自由旋转
  • [ ] 验证进料机制正常工作
  • [ ] 测试发射速度是否稳定
  • [ ] 检查所有螺钉和连接
  • [ ] 确认编码器读数准确

周期检查 (每周) ​

  • [ ] 清洁飞轮和接触面
  • [ ] 检查皮带或链条张力
  • [ ] 验证PID调参是否有效
  • [ ] 测试发射精度和一致性

飞轮系统维护 ​

检查清单 ​

查看飞轮状态:

  • 是否有裂纹或损伤
  • 表面是否光滑均匀
  • 是否有不平衡现象
  • 轴承是否有异响

检查电机:

检查项目:
✓ 接线是否牢固
✓ 冷却风扇是否工作
✓ 电机温度是否正常 (< 80°C)
✓ 是否有烧焦味或异响

调速调参 ​

调整飞轮转速:

java
// 在Robot代码中调参示例
double targetRPM = 3500;
double currentRPM = launcherMotor.getSelectedSensorVelocity();

if (currentRPM < targetRPM - 100) {
    // 加速
    output += 0.05;
} else if (currentRPM > targetRPM + 100) {
    // 减速
    output -= 0.05;
}

launcherMotor.set(ControlMode.PercentOutput, output);

进料系统 ​

常见问题 ​

问题原因解决方案
无法进料电机无电或卡死检查电源和机械连接
进料不均匀驱动皮带松动调整皮带张力
卡球现象进料通道堵塞清洁进料通道
噪声过大轴承磨损或缺油清洁或更换轴承

维护步骤 ​

  1. 清洁进料通道

    • 断开电源
    • 移除所有球
    • 用刷子清洁内部
    • 检查是否有损坏

  2. 调整皮带张力

    • 可以轻轻按压皮带，下压距离应为5-10mm
    • 过松导致打滑，过紧导致轴承损伤
    • 使用张力规或专业工具精确调整

  3. 电机轴承润滑

    • 不需要频繁润滑（通常一个赛季一次）
    • 使用轻机油或指定润滑剂
    • 不要过度润滑

传感器校准 ​

光电传感器 ​

安装位置: 进料通道出口
作用: 检测球是否装弹

校准步骤:
1. 将传感器暴露在光线下，记录模拟值
2. 打开传感器前面的灯光，记录新值
3. 在代码中设置阈值为两者的中点

接近传感器 ​

检查发射器就绪状态：

java
double sensorReading = proximitySensor.getAverageVoltage();
boolean readyToLaunch = sensorReading > THRESHOLD;

常见故障排除 ​

发射不稳定 ​

症状: 发射速度变化大，精度差

诊断:

  1. 检查飞轮是否平衡
  2. 验证PID参数
  3. 检查电源稳定性
  4. 测试电机效率

解决方案:

  • 动平衡飞轮
  • 重新调整PID参数
  • 检查电路连接
  • 必要时更换电机

发射延迟 ​

症状: 从命令到发射有显著延迟

原因:

  • 进料机制卡顿
  • 软件延迟或缓冲
  • 电机响应缓慢

验证方法:

java
long startTime = System.currentTimeMillis();
launcher.launch();
// 等待发射完成
long elapsedTime = System.currentTimeMillis() - startTime;
SmartDashboard.putNumber("Launch Delay", elapsedTime);
