💻 代码仓库 ​

主要仓库 ​

  • Robot Code - 主机器人程序

    • 分支: main (稳定), develop (开发), features/*
    • 使用WPILib 2024版本

  • Vision System - 摄像头识别

    • OpenCV C++ 实现
    • 实时目标追踪

文件结构 ​

robotcode/
├── src/           # Java源代码
├── build.gradle   # Gradle配置
├── docs/          # API文档
└── test/          # 单元测试

Git工作流 ​

bash
# 克隆仓库
git clone https://github.com/team-6353/robotcode.git

# 创建功能分支
git checkout -b feature/new-feature

# 提交和推送
git commit -m "Add new feature"
git push origin feature/new-feature

# 创建Pull Request
